# ICC Men's T20 World Cup 2022 Data Analysis

This project presents a comprehensive data analysis of the ICC Men's T20 World Cup 2022 using Python and modern data visualization techniques. The goal is to uncover meaningful insights, hidden patterns, and performance trends from tournament data through exploratory data analysis (EDA), statistical analysis, and interactive visualizations.

## Project Objectives

* Analyze team performance throughout the tournament.
* Evaluate player contributions and Player of the Match performances.
* Investigate the impact of toss decisions on match outcomes.
* Compare batting-first and chasing strategies.
* Examine score distributions and winning margins.
* Calculate win probabilities under different match scenarios.
* Discover venue-specific trends and advantages.
* Identify key factors influencing match results.

## Technologies Used

* Python
* NumPy
* Pandas
* Plotly
* Matplotlib
* Seaborn

## Key Analyses

### Team Performance Analysis

* Matches played by each team
* Total wins and win percentages
* Tournament progression and stage-wise performance

### Player Statistics Analysis

* Player of the Match awards
* Top-performing players
* Impactful individual performances

### Match Outcome Analysis

* Toss winner vs match winner relationship
* Batting first vs bowling first success rates
* Winning margin distribution

### Score Analysis

* First innings and second innings score distributions
* High-scoring and low-scoring match patterns
* Score thresholds associated with winning probabilities

### Venue Analysis

* Match distribution across venues
* Venue-specific winning trends
* Impact of venue conditions on match outcomes

### Predictive Insights

* Win probability based on first innings scores
* Chasing success rates
* Score ranges associated with higher chances of victory

## Interactive Visualizations

The project leverages Plotly to create interactive and publication-ready visualizations, enabling users to:

* Explore data dynamically
* Filter and inspect trends
* Compare team and player performances
* Understand tournament patterns through engaging visual storytelling

## Project Outcome

This analysis transforms raw match data into actionable cricket insights, helping analysts, cricket enthusiasts, and data science learners better understand the factors that influenced team success and match outcomes during the ICC Men's T20 World Cup 2022.
